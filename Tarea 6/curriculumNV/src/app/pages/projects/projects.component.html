Social Network simulation with Linked Lists on C language (2021)

  • Implemented a linked list with C in order to use that library to simulate a social network. Each user was a node, and with every follow, the linked list incremented its size
  • Had basic operations like following other users, which linked a node with another, stop following someone, which deleted a node from your main linked list.

Implementation of Huffman Tree in C (2021)

  • Implemented the huffman algorithm to compress a string of text.
  • Used an implementation of a hashmap in C to make a count of every character, then used a priority queue to store each node with it's character
  • Made a father node for the first 2 nodes in PQ and made another PQ with them to traverse and get the compressed string.

Chess game on Java with graphic interface (2022)

  • Did a player vs player experience of a chess game in Java with a graphic interface
  • Utilized a singleton class for the board, made an abstract class named "Piece", and from that class every piece had it's own class that inherited the abstract methods.
  • Used a linked list to track every piece of each side of the board
